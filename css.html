2. การเขียน css มีกี่แบบ

รูปแบบ คำอธิบาย
1. Inline css  เขียนภายใน HTML
2. external css  นำไฟล์ css เข้าจากภายนอก
3. internal css  เขียนภายใน Element html

3. Selector มีกี่แบบอะไรบ้าง

รูปแบบ คำอธิบาย
1. Id  การกำหนดชื่อของ element โดยจะมีได้แค่ element เดียวเท่านั้น
2. Class  การกำหนดชื่อของ element โดยสามารถมีได้หลาย element
3. Pseudo element  การกำหนดเฉพาะบางส่วนของ element
4. Pseudo class  การกำหนดเฉพาะบางสถานะของ class
5. Combinators  การกำหนดเงื่อนระหว่าง 2 selector
6. Attribute selector  การกำหนดการเลือกชื่อ attribute ใน element

4. combinator มีกี่แบบอะไรบ้าง

รูปแบบ คำอธิบาย
1. descendant selector (space)  a b = tag b ทั้งหมดที่อยู่ใน a
2. child selector (>)  a > b = tag b ที่เป็นลูกของ a
3. adjacent sibling selector (+)  a + b = tag b ที่อยู่ล่าง a และติดกับ a เท่านั้น
4. general sibling selector (~)  a ~ b = tag b ที่อยู่ล่าง a ทุก tag
